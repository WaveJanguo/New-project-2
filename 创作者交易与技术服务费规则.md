# 创作者交易与技术服务费规则

更新时间：2026-04-30

## 1. 内容规则

平台内容分为两类：

- 源站收录内容：来自 `EvoLinkAI/awesome-gpt-image-2-prompts` 的公开案例，本站全部免费复制，并保留来源链接。
- 创作者原创内容：由创作者上传、设置价格并出售，买家付款后获得完整提示词。

## 2. 交易规则

创作者可以：

- 上传原创提示词。
- 上传效果图。
- 设置单条价格。
- 查看订单和收入。
- 申请提现。

买家可以：

- 免费复制源站收录内容。
- 购买创作者原创提示词。
- 查看已购买内容。
- 收藏提示词。

## 3. 技术服务费

平台按每笔创作者订单收取技术服务费。

当前规则：

- 技术服务费率：10%。
- 服务费从订单金额中扣除。
- 买家支付金额不额外加价。
- 创作者收入 = 订单金额 - 技术服务费。

示例：

- 提示词售价：19.90 元。
- 平台技术服务费：1.99 元。
- 创作者收入：17.91 元。

## 4. 上线需要补齐

真实交易上线前需要开发：

- 账号系统。
- 创作者认证。
- 提示词审核。
- 订单系统。
- 支付回调。
- 结算和提现。
- 退款规则。
- 侵权投诉与下架机制。
