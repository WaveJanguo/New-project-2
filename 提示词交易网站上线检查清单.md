# 提示词交易网站上线检查清单

更新时间：2026-04-30

## 1. 内容检查

- 至少准备 70 条以上提示词。
- 每条提示词有标题。
- 每条提示词有完整正文。
- 每条提示词至少有 1 张效果图。
- 每条提示词有分类。
- 每条提示词有权限类型：免费或会员。
- 第三方来源内容保留来源链接。
- 高风险版权内容已剔除。
- 违规、色情、暴力、侵权内容已剔除。

## 2. 前台功能检查

- 首页正常加载。
- 图片正常显示。
- 搜索功能正常。
- 分类筛选正常。
- 免费提示词可以复制。
- 会员提示词对非会员锁定。
- 会员弹窗能打开和关闭。
- 登录入口正常。
- 我的页面正常。
- 移动端布局正常。
- 不出现横向滚动条。

## 3. 会员与支付检查

- 会员价格显示为 9.9 元。
- 未登录用户购买前会进入登录。
- 创建订单成功。
- 支付二维码或支付跳转正常。
- 支付成功后订单状态更新。
- 支付成功后会员状态更新。
- 刷新页面后会员仍有效。
- 支付失败有提示。
- 重复支付回调不会重复处理。
- 未支付订单不会开通会员。

## 4. 后台检查

- 管理员可以登录。
- 管理员可以新增提示词。
- 管理员可以编辑提示词。
- 管理员可以上传图片。
- 管理员可以上架提示词。
- 管理员可以下架提示词。
- 下架内容前台不可见。
- 管理员可以查看订单。
- 管理员可以查看用户。

## 5. 安全检查

- 会员提示词完整正文不在前端源码中暴露。
- 后台接口需要管理员权限。
- 支付回调校验签名。
- 用户只能查看自己的订单。
- 用户只能管理自己的收藏。
- 密码或 Token 不写入前端代码。
- 上传图片限制类型和大小。
- API 有基础频率限制。

## 6. 性能检查

- 图片开启懒加载。
- 图片有压缩版本。
- 首页首屏加载时间可接受。
- 搜索和筛选响应迅速。
- 移动端图片不会撑破布局。
- 静态资源开启缓存。

## 7. SEO 检查

- 首页有合理 title。
- 首页有 description。
- 分类页有独立 title。
- 提示词详情页有独立 title。
- 图片有 alt。
- 页面 URL 可读。
- 生成 sitemap。
- 配置 robots.txt。

## 8. 法务与合规检查

- 有用户协议。
- 有隐私政策。
- 有支付说明。
- 有退款规则。
- 有侵权投诉方式。
- 有联系方式。
- 国内上线准备备案。

## 9. 数据监控

- 接入访问统计。
- 记录提示词浏览。
- 记录复制行为。
- 记录会员弹窗触发。
- 记录支付转化。
- 记录错误日志。
- 支付异常有告警。

## 10. 上线前最终确认

- 域名解析正确。
- HTTPS 正常。
- 支付回调地址是线上地址。
- 数据库已备份。
- 管理员账号已设置。
- 首批内容已审核。
- 客服或联系方式可用。
- 关键页面在手机和电脑上都检查过。
