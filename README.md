# GPT Image 2 提示词网站

基于 `EvoLinkAI/awesome-gpt-image-2-prompts` 的提示词交易站原型。

## 功能

- 收录源仓库 `cases/*.md` 中可解析到完整 Prompt 的 312 条提示词
- 首页展示提示词交易卡片流
- 支持提示词详情弹窗，展示大图、完整提示词、作者和来源链接
- 源站收录提示词全部免费复制
- 支持创作者原创提示词交易演示
- 支持关键词、分类、语言、免费/付费筛选
- 支持本地收藏提示词

## 运行

直接启动一个静态服务器：

```bash
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## 数据来源

- 本地数据文件：`data/prompts.json`
- 可直接打开页面的数据脚本：`data/prompts-data.js`
- 原始仓库：<https://github.com/EvoLinkAI/awesome-gpt-image-2-prompts>

## 同步源站数据

```bash
node scripts/sync-source-prompts.js
```

该脚本会解析源仓库英文分类文件，生成 `data/prompts.json` 和 `data/prompts-data.js`。
